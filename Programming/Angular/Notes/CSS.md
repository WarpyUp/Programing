
### :host

Якщо нам потрібно стилізувати сам "хост" в батьківському класі нашого компонента, тобто тег `<app-child-component>`. Для цього в Angular існує спеціальний псевдо-клас `:host`, пишемо його в дочірньому компоненті.
```css
:host{
	display: block;
	width: 600px;
}
```

### !important

`!important` змушує браузер застосувати ваш стиль, ігноруючи майже всі інші. Використовувати `!important` вважається поганою практикою.
```css
input {
    font-weight: normal !important;
}
```
